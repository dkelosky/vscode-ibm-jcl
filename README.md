# ibm-jcl README

Extension "ibm-jcl".

## Features

Basic highlighting for IBM mainframe Job Control Language (JCL).

> Tip: Add editor configuration in user.settings to highlight continuation column, e.g. `"editor.rulers": [71, 72, 80],`
